HTML

  • head contains meta data
  • meta data helps optimise searches for search engines
  • body contains all the funky stuff
  • header is the top of the page
  • fotter is at the bottom, contains copywrite or signatures
  • h1-6 are headings
  • ul and li are lists
  • p are for paragraphs

CSS

  • * margin then padding
  • header, footer add width, height, background-color, and color
  • CSS is the style of the web page and can be done in many ways
  • Must link the CSS file to the HTML code using link command
  • rel is the relationship of the CSS file whiles href is the URL/Directory of where the CSS file is located

GIT

  • code . - Opens VS Code
  • git status - displays current branch
  • git checkout -b ADDNAME/ADDNAME - Adds Branch to main
  • git add -A - Adding changes made to current branch
  • git commit -m "COMMENT" - Merging with main branch
  • git pull origin main - Downloads up to date data for main branch
  • git push origin ADDNAME/ADDNAME - Uploads the changes done from current branch to main branch
  • git checkout main - GOES TO MAIN BRANCH

JavaScript

  • Order matters, starts from top to bottom
  • var is variable, defines things
  • functions work similar to var but contains more complex code
  • console.log() allows display of words if conditions are met
  • if statements can be used if there are different things to select
  • make sure to include semicolons after brackets to split statements
